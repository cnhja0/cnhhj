04 · Próximo Passo

Prioridade #1: Página de Edição de Perfil

★ Próxima implementação imediata

A aba MAIS já tem o item "Perfil" no menu, mas atualmente leva a um TODO. A próxima task é construir essa tela.

Spec funcional

A tela permite ao instrutor editar seus dados pessoais e profissionais. Deve ser acessível por:

  • Toque no item "Perfil" na aba MAIS
  • Toque no avatar/header da aba MAIS (futuro: também no avatar da Home)

Layout planejado

Estrutura visual

  • AppBar com botão "Voltar" + título "Editar perfil"
  • Avatar grande centralizado com botão circular preto sobreposto (ícone câmera) para trocar foto
  • Cards agrupando: Dados pessoais, Contato, Sobre você
  • Botão Salvar no rodapé fixo (sticky) com scale-on-press
  • Botão "Cancelar" secundário ao lado do Salvar
  • Indicador de "alterações não salvas" se o usuário sair sem salvar

Aspectos técnicos

  • Arquivo: lib/features/profile/profile_edit_screen.dart
  • Controller: profile_edit_controller.dart com Riverpod (Notifier)
  • Rota: /profile/edit em AppRoutes.profileEdit (já existe a constante)
  • Repositório: InstructorRepository.updateProfile() (já implementado no mock)
  • Após salvar: invalidar currentInstructorProvider + Profile provider — Home reflete o nome novo
  • Foto: CnhhjPhotoPicker já existe; reutilizar
  • Componentes: CnhhjTextField, CnhhjDateField, CnhhjDropdown, CnhhjAvatar, CnhhjPrimaryButton, CnhhjSecondaryButton, TabHeader (modo "Editar")

Estimativa

~2 horas de implementação · ~250 linhas Dart · Não bloqueia outras frentes
